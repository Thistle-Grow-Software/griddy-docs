# Griddy Platform

Welcome to the unified documentation for the **Griddy** platform — a suite of tools
for working with NFL and American football data.

## Projects

<div class="grid cards" markdown>

-   **Griddy SDK Python**

    ---

    A Python SDK providing programmatic access to several American football data
    sources, including NFL.com APIs, Next Gen Stats, and Pro Football Reference.

    [:octicons-arrow-right-24: Go to docs](griddy-sdk-python/index.md)

-   **fbcm**

    ---

    Command-line tools for downloading and archiving football content from NFL Plus,
    YouTube, and other sources. Generates metadata for Plex and Jellyfin.

    [:octicons-arrow-right-24: Go to docs](fbcm/index.md)

-   **Griddy Archive Manager**

    ---

    A Django application for cataloging and managing football game video archives.
    Tracks games, teams, venues, and video assets across multiple football levels.

    [:octicons-arrow-right-24: Go to docs](griddy-archive-manager/index.md)

-   **Griddy Web Portal**

    ---

    The first consumer-facing client — a React web portal for authenticated coaches,
    scouts, and media to browse stats and watch game film. Currently architecture
    decisions only; application code lands in subsequent stories.

    [:octicons-arrow-right-24: Go to docs](griddy-web-portal/index.md)

</div>
