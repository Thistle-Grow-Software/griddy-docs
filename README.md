# griddy-docs

Unified documentation site for the Griddy platform. Merges docs from all Griddy subprojects into a single site using MkDocs with the [mkdocs-multirepo-plugin](https://github.com/jdoiro3/mkdocs-multirepo-plugin).

## Quick Start

```bash
uv sync
uv run mkdocs serve
```

## Projects

- [griddy-sdk-python](https://github.com/Thistle-Grow-Software/griddy-sdk-python) - Python SDK for NFL data
- [fbcm](https://github.com/Thistle-Grow-Software/fbcm) - CLI tool for downloading football content
- [griddy-archive-manager](https://github.com/Thistle-Grow-Software/griddy-archive-manager) - Django app for cataloging football game video archives
